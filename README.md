# 🌿 nometa.xyz
| Не пишите мета-сообщения. Просто задайте вопрос.

Минималистичный одностраничный сайт о культуре общения в чатах, мессенджерах и на форумах. Объясняет, почему фразы вроде «Привет, можно спросить?» тратят время, и показывает, как писать сообщения, на которые хочется отвечать.


✨ Возможности
🌍 Два языка — мгновенное переключение RU / EN без перезагрузки

🎨 Спокойная тёмно-зелёная тема — мягкие тона, приглушённые акценты, комфортное чтение

📖 Жизненные примеры — работа, учёба, путешествия, медицина, быт (не только для IT)

📊 Полоса прогресса чтения — плавный индикатор вверху страницы

🌀 Ненавязчивые анимации — fade-in при скролле, дышащие фоны, лёгкие ховеры

📦 Один файл — index.html, ноль зависимостей, ноль сборки

📱 Адаптивность — корректно работает на телефонах, планшетах и десктопах
